**ngx-atomic-i18n** の概要です。ここから始めましょう。

## インストール

好みのパッケージマネージャーで導入します。

### npm

```bash
npm install ngx-atomic-i18n
```

### pnpm

```bash
pnpm add ngx-atomic-i18n
```

## 最初のステップ

最初の翻訳バンドルを作成してアプリに組み込みます。

### 例

```ts
import { provideTranslation } from 'ngx-atomic-i18n';
```
