歡迎使用 **ngx-atomic-i18n**。這是快速入門與整體概覽。

## 安裝

使用你習慣的套件管理工具。

### npm

```bash
npm install ngx-atomic-i18n
```

### pnpm

```bash
pnpm add ngx-atomic-i18n
```

## 第一個步驟

建立你的第一個翻譯資源並接上應用程式。

### 範例

```ts
import { provideTranslation } from 'ngx-atomic-i18n';
```
